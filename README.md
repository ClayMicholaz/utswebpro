🔎 Lost and Found Web Application

A simple Lost and Found Web Application built for the Mid-Term Web Programming Exam.
This application helps users report lost items and post found items so they can reconnect items with their owners.

The system allows users to submit item information through a web interface and browse listings to find matching items.

📌 Project Description

The Lost and Found App is a web-based system designed to help people report and recover lost belongings. Users can:

Report lost items

Report found items

View lists of reported items

Contact the person who reported the item

This application was built using HTML, CSS, JavaScript, PHP, and Bootstrap.
